Texture & Depth Recipe

不用 drop shadow — 用这些手法做"质感"

Thick border 3-4px solid, 比填充色深 2 档
Bottom shadow bar box-shadow: 0 3-4px 0 [darker-color]
Inner top highlight inset 0 2px 0 rgba(255,255,255,0.5-0.8)
Surface shine ::after pseudo, top 40%, white→transparent gradient
Inner white border ::before pseudo, 2px solid rgba(255,255,255,0.5), inset 4px
Text depth text-shadow: 0 2-3px 0 [dark], NOT blur

Design Notes

vs v1 变了什么：

  • 去掉 drop shadow，改为 底部实色条 + 内高光 + 顶部光泽
  • 去掉粉色 accent，候选：番茄红 / 橘色 / 紫莓 / 亮金
  • 字体更粗 — 提供 6 种对比
  • Panel 有 分层结构（外描边 → 内白描边 → 内容），参照 GUI pack 的窗口
  • 加入了 装饰形状库：星星、徽章、缎带、硬币、爱心、旗帜
  • 按钮有 按压底边 + 顶部高光弧（candy button 风格）
  • Progress bar 有 内陷阴影 + 填充高光条
